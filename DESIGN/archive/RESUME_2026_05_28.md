A multiplayer online game (MUD) built on the Evennia/Django engine, with XRPL blockchain and AI/LLM integration. One major goal has been to engineer a genuinely decoupled system — distinct game subsystems interact only through well-defined interfaces, enforced via bespoke Python libraries.

**Key technical work:**

- XRPL on-chain integration — supporting APIs for multisig transactions and NFT metadata

- LLM integration via OpenRouter and OpenAI APIs, including creation and consumption of embeddings stored in vector databases

- Horizontal scaling framework: engineered a solution to distribute Evennia's single-threaded Twisted Reactor across multiple instances sharing a single vertically-scaled Postgres instance

- XRPL wallet management interface (private repo) for internal operational use

- LLM test-bot harness (private repo) for automated game dialogue testing

- Python library development with a focus on minimal coupling between game systems
